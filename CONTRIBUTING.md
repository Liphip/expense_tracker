## Getting Started

### Local Deployment

To deploy the app locally:

1. Clone the repository: `git clone https://github.com/Liphip/expense_tracker.git`
2. Navigate to the project directory: `cd expense_tracker`
3. Open the `index.html` file in your browser.

## Issues
